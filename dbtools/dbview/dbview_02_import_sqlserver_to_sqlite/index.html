Importing SQL Server data into a SQLite database

Open a SQLite database and run the following import command (replace example parameter values with your own values)

import

  src_type           = 'sqlserver'
  src_server         = 'REPLACE_ME_SQL_ServerName_Instance'
  src_database       = 'REPLACE_ME_SQL_Database'

  tgt_table          = 'REPLACE_ME_target_table_name'
  tgt_exists_action  = 'fail'

select * from dbo.Table1
;
parameters type required description/notes
src_type string required set to 'sqlserver'
src_server string required name of source SQL Server (and instance if needed)

e.g. 'Altair\Prod'
src_database string required name of source SQL Server database
tgt_table string required name of target table in current SQLite database into which data should be loaded
tgt_exists_action string optional action to take if the specified target table already exists
valid values are 'replace' or 'fail'

default is 'fail' if not defined
